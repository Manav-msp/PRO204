# PRO204

The classification script first parsed each line of the Spectrum Lab event log using a regular expression to extract the date, time, signal amplitude, noise floor, peak frequency, and duration. Events with a peak frequency outside the 1180–1470 Hz range or with a zero signal value were discarded. The signal-to-noise ratio was then computed for each remaining event by subtracting the noise floor from the signal amplitude. Events with an SNR below 18 dB were removed. The remaining events were passed to a classification function that assigned one of four labels based on duration: events lasting 1–5 loops were labelled as underdense meteors, 6–30 loops as overdense meteors, 31–80 loops as satellites, and anything above 80 loops as aircraft. Both meteor categories were then combined into a single confirmed meteor count for the hourly aggregation. The classified events were written to a CSV file and grouped by hour to produce the hourly rate table. Five plots were then generated: a three-panel hourly activity bar chart, a duration versus SNR scatter plot, a frequency drift plot over time, an SNR histogram, and an hour-of-day versus date heatmap.
